# Create Your First Competition

In this tutorial, we will create a complete competition on WodSmith. By the end, you will have a live competition page with divisions configured, events added, and registration open to athletes.

## What We'll Accomplish

1. Create a competition
2. Configure divisions
3. Add workout events
4. Set up registration
5. Open to athletes

This process takes about 15-20 minutes. Let's begin!

## Step 1: Create the Competition

First, we'll create the competition and set the basic details.

1. Navigate to **Competitions** in the sidebar
2. Click **New Competition**
3. Enter the competition name (e.g., "Spring Throwdown 2025")
4. Set the competition date
5. Enter your location details
6. Click **Create**

**You should see** your competition dashboard. This is your control center for managing the event.

## Step 2: Configure Divisions

Now we'll set up the divisions athletes can register for.

1. Click **Divisions** in the competition menu
2. Click **Add Division**
3. Enter "RX Male" as the division name
4. Set the participant cap (e.g., 20)
5. Click **Save**

**Notice** the division appears in your list.

### Add More Divisions

Let's add the remaining standard divisions:

1. Click **Add Division** again
2. Create "RX Female" with a cap of 20
3. Add "Scaled Male" with a cap of 15
4. Add "Scaled Female" with a cap of 15

**You should see** four divisions listed:
- RX Male (0/20)
- RX Female (0/20)
- Scaled Male (0/15)
- Scaled Female (0/15)

## Step 3: Add Competition Events

Now we'll add the workouts athletes will compete in.

1. Click **Events** in the competition menu
2. Click **Add Event**
3. Enter "Event 1 - For Time"
4. Select **For Time** as the workout type

### Configure the Workout

1. Add movements:
   - 21 Thrusters (95/65 lb)
   - 21 Pull-ups
   - 15 Thrusters
   - 15 Pull-ups
   - 9 Thrusters
   - 9 Pull-ups
2. Set a time cap of 10 minutes
3. Click **Save**

**You should see** Event 1 appear in your events list.

### Add a Second Event

1. Click **Add Event**
2. Enter "Event 2 - AMRAP"
3. Select **AMRAP** as the workout type
4. Set time cap: 8 minutes
5. Add movements:
   - 10 Box Jumps (24/20")
   - 10 Toes-to-Bar
6. Click **Save**

**Notice** you now have two events scheduled.

## Step 4: Configure Registration

Let's set up how athletes will register.

1. Click **Registration** in the competition menu
2. Set the registration open date
3. Set the registration close date
4. Toggle **Require Waiver** to on

### Set Entry Fees (Optional)

1. Click **Pricing**
2. Enter the registration fee (e.g., $50)
3. Connect your payment account if prompted
4. Click **Save**

**You should see** registration settings confirmed.

## Step 5: Review and Publish

We're ready to make the competition live!

1. Click **Settings** in the competition menu
2. Review all details:
   - Competition info
   - Divisions configured
   - Events added
   - Registration ready
3. Toggle **Published** to on
4. Click **Save**

**You should see** the competition status change to **Live**.

## Step 6: Share Your Competition

Your competition is now live! Let's get the registration link.

1. Click **Share** or find the public URL
2. Copy the competition link

**Notice** the URL format: `app.wadsmith.com/c/your-competition-slug`

This is the link you'll share with athletes to register.

## You've Done It!

Congratulations! You have successfully created a competition with:

- 4 divisions configured
- 2 workout events ready
- Registration open for athletes
