# Managing Teams

Learn how to set up and manage your team in WodSmith.

## Setting Up Your Team

### Initial Setup

When you first create a team:

1. **Enter team details**
   - Team name (your gym name)
   - Description (optional)
   - Timezone

2. **Upload branding** (optional)
   - Team logo
   - Cover image

3. **Configure settings**
   - Default visibility for workouts
   - Member permissions

## Inviting Team Members

### Send Invitations

1. Go to **Settings** → **Members**
2. Click **Invite Member**
3. Enter the email address
4. Select their role:
   - **Admin** - Full access
   - **Coach** - Programming access
   - **Member** - Athlete access
5. Click **Send Invitation**

### Bulk Invitations

For multiple members:

1. Click **Bulk Invite**
2. Enter emails (one per line)
3. Select default role
4. Send all invitations

### Invitation Status

Track pending invitations:
- **Pending** - Invitation sent, not accepted
- **Accepted** - Member has joined
- **Expired** - Invitation expired (resend available)

## Managing Member Roles

### Changing Roles

1. Go to **Settings** → **Members**
2. Find the member
3. Click the role dropdown
4. Select new role

### Role Capabilities

| Action | Admin | Coach | Member |
|--------|:-----:|:-----:|:------:|
| View workouts | ✓ | ✓ | ✓ |
| Log scores | ✓ | ✓ | ✓ |
| Create workouts | ✓ | ✓ | ✗ |
| Manage programming | ✓ | ✓ | ✗ |
| Invite members | ✓ | ✗ | ✗ |
| Manage billing | ✓ | ✗ | ✗ |
| Delete team | ✓ | ✗ | ✗ |

## Removing Members

### Remove a Member

1. Go to **Settings** → **Members**
2. Find the member
3. Click the **...** menu
4. Select **Remove from team**
5. Confirm removal

### What Happens When Removed

- Member loses access immediately
- Their score history is preserved
- They can be re-invited later

## Team Settings

### General Settings

Configure your team's basic information:

- **Name** - Your gym's display name
- **Slug** - URL-friendly identifier
- **Description** - Brief description
- **Timezone** - Default for scheduling

### Privacy Settings

Control visibility:

- **Public workouts** - Visible to non-members
- **Private workouts** - Members only
- **Member directory** - Show/hide member list

## Multiple Locations

For gyms with multiple locations:

### Option 1: Separate Teams

Create a team for each location:
- Independent programming
- Separate member lists
- Individual billing

### Option 2: Single Team

Use one team with tags:
- Shared programming
- Location tags for filtering
- Unified billing

## Transferring Ownership

To transfer team ownership:

1. Ensure new owner has Admin role
2. Contact support for ownership transfer
3. Verify transfer with both parties

## Team Analytics

View team statistics:

- Active members
- Workout completion rates
- Most popular workouts
- Member engagement trends

Access analytics from **Settings** → **Analytics**.

## Best Practices

### For New Teams

1. Set up team profile completely
2. Invite coaches first
3. Create initial programming
4. Then invite members

### Ongoing Management

- Regularly review member list
- Archive inactive members
- Keep roles up to date
- Monitor engagement

## Troubleshooting

### Member Can't Access Team

1. Verify invitation was accepted
2. Check member role
3. Confirm team visibility settings

### Need to Recover Access

Contact support with:
- Team name
- Your email
- Proof of ownership
