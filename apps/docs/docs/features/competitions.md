# Competitions

Run throwdowns and competitions with full event management.

## Competition Overview

WodSmith supports various competition formats:

- **In-house throwdowns** - Gym competitions
- **Multi-gym events** - Regional competitions
- **Online competitions** - Remote participation

## Events & Workouts

Add competition events:

1. Create or select workouts
2. Assign to competition events
3. Configure scoring and tiebreakers
4. Set time caps and standards

## Registration

### Athlete Registration

Athletes register through:
- Public registration page
- Direct invitation links
- Team-based registration

### Waivers

Collect digital waivers during registration:
- Custom waiver text
- Electronic signatures
- Automatic record keeping

## Heat Management

Organize athletes into heats:

1. View registered athletes by division
2. Auto-generate heat assignments
3. Adjust manually as needed
4. Print/export heat sheets

## Scoring & Leaderboards

### Live Scoring

- Judges enter scores in real-time
- Automatic ranking calculations
- Tiebreaker application

### Leaderboards

- Overall standings by division
- Event-by-event results
- Public leaderboard pages

## Results & Awards

After the competition:

- Final standings verification
- Export results
- Generate certificates/awards
- Post results to social media
