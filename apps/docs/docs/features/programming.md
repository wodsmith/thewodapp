# Programming

Build and schedule workout programs for your members.

## Programming Tracks

Tracks are named programs that organize workouts for specific goals:

- **Daily WOD** - Your gym's main daily workout
- **Competition Prep** - Advanced programming for competitors
- **Foundations** - Beginner-friendly scaled programming
- **Open Gym** - Optional additional work

## Creating a Track

1. Go to **Programming** â†’ **Tracks**
2. Click **New Track**
3. Configure track settings:
   - Name and description
   - Visibility (public or members-only)
   - Color for calendar display

## Scheduling Workouts

Assign workouts to specific dates:

1. Open your track
2. Navigate to the desired date
3. Click **Add Workout**
4. Select or create a workout
5. Set the scheduled time

### Bulk Scheduling

For recurring programs:

1. Create a **Template** with your weekly structure
2. Apply the template to date ranges
3. Adjust individual days as needed

## Calendar View

The calendar provides an overview of scheduled programming:

- **Day view** - Detailed single-day schedule
- **Week view** - Weekly programming at a glance
- **Month view** - Monthly overview

## Publishing

Control when athletes see upcoming workouts:

- **Draft** - Only visible to coaches
- **Published** - Visible to all team members
- **Scheduled publish** - Automatically publish at a set time

## Templates

Save time with reusable templates:

1. Create a workout structure
2. Save as template
3. Reuse across multiple tracks and dates

Templates are perfect for:
- Weekly programming patterns
- Recurring strength cycles
- Competition prep phases
