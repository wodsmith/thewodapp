# How to Schedule Programming

Add workouts to your gym's training calendar.

## Prerequisites

- Coach or Admin role on your team
- At least one workout created

## Scheduling a Single Workout

1. Navigate to **Programming**
2. Click on the target date
3. Click **Add Workout**
4. Select the workout from your library
5. Set the scheduled time
6. Click **Save**

## Creating Programming Tracks

Organize workouts into named tracks:

1. Go to **Programming** → **Tracks**
2. Click **New Track**
3. Enter track name (e.g., "Daily WOD", "Competition Prep")
4. Set visibility and color
5. Click **Create**

## Bulk Scheduling with Templates

For recurring programs:

1. Create a **Template** with your weekly structure
2. Go to **Programming** → **Apply Template**
3. Select the template
4. Choose date range
5. Click **Apply**

## Publishing Scheduled Workouts

Control when athletes see upcoming programming:

| Action | Result |
|--------|--------|
| **Keep as Draft** | Visible only to coaches |
| **Publish Now** | Immediately visible |
| **Schedule Publish** | Auto-publish at set time |

## Calendar Views

Switch between views:

- **Day** - Detailed single-day schedule
- **Week** - Weekly overview
- **Month** - Monthly planning view
