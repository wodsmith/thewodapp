# How to Manage Team Members

Invite members, assign roles, and manage your team roster.

## Prerequisites

- Admin role on your team (for invitations and role changes)
- Coach role can view members

## Inviting New Members

1. Go to **Settings** → **Members**
2. Click **Invite Member**
3. Enter the email address
4. Select role: **Admin**, **Coach**, or **Member**
5. Click **Send Invitation**

## Bulk Invitations

For multiple members:

1. Click **Bulk Invite**
2. Enter emails (one per line)
3. Select default role
4. Click **Send All**

## Changing Member Roles

1. Go to **Settings** → **Members**
2. Find the member
3. Click the role dropdown
4. Select the new role

## Role Capabilities

| Permission | Admin | Coach | Member |
|------------|:-----:|:-----:|:------:|
| View workouts | ✓ | ✓ | ✓ |
| Log scores | ✓ | ✓ | ✓ |
| Create workouts | ✓ | ✓ | ✗ |
| Manage programming | ✓ | ✓ | ✗ |
| Invite members | ✓ | ✗ | ✗ |
| Manage billing | ✓ | ✗ | ✗ |

## Removing Members

1. Go to **Settings** → **Members**
2. Find the member
3. Click **...** → **Remove from team**
4. Confirm removal

The member loses access immediately. Score history is preserved.
