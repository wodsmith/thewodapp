# How to Create a Workout

Create workouts of any type in WodSmith.

## Prerequisites

- Coach or Admin role on your team

## Creating an AMRAP

1. Navigate to **Workouts** → **New Workout**
2. Select **AMRAP**
3. Set the time cap in minutes
4. Add movements and rep counts
5. Add scaling options if needed
6. Click **Publish** (or **Save as Draft**)

## Creating a For Time Workout

1. Navigate to **Workouts** → **New Workout**
2. Select **For Time**
3. Add movements with rep schemes
4. Set a time cap (optional but recommended)
5. Specify RX and scaled weights
6. Click **Publish**

## Creating an EMOM

1. Navigate to **Workouts** → **New Workout**
2. Select **EMOM**
3. Set total duration
4. Add movements for each minute
5. Click **Publish**

## Creating a Strength Session

1. Navigate to **Workouts** → **New Workout**
2. Select **Strength**
3. Add lifts with sets and reps
4. Choose scoring type:
   - Max weight
   - Total volume
   - Check-in only
5. Click **Publish**

## Adding Scaling Options

For any workout type:

1. Click **Add Scaling**
2. Name the scale (e.g., "Scaled", "Foundations")
3. Specify movement substitutions or weight reductions
4. Repeat for additional scaling levels

## Saving and Publishing

| Status | Visibility |
|--------|------------|
| **Draft** | Only coaches |
| **Published** | All team members |
