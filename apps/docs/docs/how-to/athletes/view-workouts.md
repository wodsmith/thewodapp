# How to View Workout Standards

Review workout details, movement standards, and scaling options before competition.

## Steps

1. Navigate to the competition page
2. Click **Workouts** or **Events**
3. Select the workout you want to review

## Workout Information

Each workout page displays:

- **Workout structure** - Time cap, rep scheme, format
- **Movements** - All movements with rep counts
- **Standards** - Movement requirements and no-reps
- **Equipment** - Required equipment and weights
- **Scaling** - Division-specific modifications

## RX vs Scaled Standards

Review the standards for your division:

| Element | RX | Scaled |
|---------|-----|--------|
| Weights | Prescribed | Reduced |
| Movements | Full range | Modified |
| Equipment | Standard | Alternative |

## Before Competition Day

- Note any equipment you need to bring
- Practice unfamiliar movements
- Review movement standards videos if available
