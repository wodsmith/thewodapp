# How to View Broadcasts

See announcements from competition organizers.

## Viewing Announcements

1. Open the competition page
2. Click the **Announcements** tab

All broadcasts sent to you appear in reverse chronological order, showing the title, message body, and the date it was sent.

If the organizer hasn't posted any announcements, you'll see an empty state message.

## Email Notifications

When an organizer sends a broadcast with email enabled, you'll also receive the announcement in your inbox. The email contains the same title and message as the in-app announcement.
