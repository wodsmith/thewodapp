1. Pick the **organizing team**. Only teams approved as organizers appear — if yours is missing, complete organizer onboarding first.
2. Set the **name, dates, and timezone**. Dates display to athletes in the competition's timezone, not theirs.
3. Choose the **competition type** — this decides your event-day tooling and can't be casually flipped later:
   - **In-person** — heats, lanes, venues, and the day-of check-in kiosk.
   - **Online** — submission windows and video review instead of heats.
4. Optionally attach a **series** to aggregate scores across competitions. Picking one inherits the series' division template.
5. Create. The competition starts unpublished — athletes can't see it yet.

**Then, in order:** Divisions → Events → Pricing (connect Stripe if charging) → Waivers → publish when registration should open.

First time? The [first-competition tutorial](https://docs.wodsmith.com/tutorials/organizers/first-competition) walks the whole setup end to end.
