Scores are published per **event + division** pair, so you can release results division by division as they finalize.

**Know your defaults:**

- **Online competitions:** every result is hidden until you publish it.
- **In-person competitions:** results are visible as soon as they're entered, unless you've started using publish controls.

**To publish:**

1. Select the event and division in the results grid.
2. Enter or verify the scores.
3. Check standings on **Leaderboard Preview** — it shows everything, published or not.
4. Toggle **Publish** for that event + division.

To release an entire event at once, use the division-results quick action on the dashboard overview.

**What publishing controls:** until an event + division is published, its scores *and* athlete video links are hidden from the public leaderboard. Unpublishing hides them again — useful if you need to correct a scoring dispute quietly.

If athletes report "the leaderboard is empty," an unpublished division is almost always why.
