If you created a multi-part workout (Part A / Part B) as separate events, you can group them under a parent so they're scheduled as one block while staying scored per part.

**To group:**

1. Tick the checkbox on two or more standalone events.
2. Click **Group as one event** in the banner that appears.
3. Name the parent (e.g. "Event 2: A + B") and confirm.

The parts become sub-events under the parent, keeping their order and their existing scores. Athletes submit a score per part; leaderboards combine them per the part's scoring.

**Before grouping, note:**

- Events with **scheduled heats can't be grouped** — delete those heats first, then group, then reschedule on the parent.
- Sub-events can't be grouped again (one level only).
- The parent is published only if every selected event was already published. Publishing or unpublishing a parent always cascades to its parts — a draft parent hides all its parts from athletes.

For division-specific event lineups, manage the **parent's** division mapping on the Event Divisions page — parts follow their parent automatically.
